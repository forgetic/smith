# Start a development session

Use this checklist before editing Smith.

## Checklist

1. Read `README.md` and `AGENTS.md`.
2. Read `docs/README.md` and choose task-relevant docs.
3. Read `docs/reference/agent-lessons/README.md` and any lessons whose tags
   match the task. For provider/auth work, read all active lessons.
4. If a change touches a Temper protocol, read the corresponding Temper-owned
   docs in `../temper/docs/reference/` before editing:
   - `interactive-process-responder-protocol.md`
   - `workflow-role-decision-process-protocol.md`
   - `interactive-conversation-interface.md`
5. Use `rg` to discover related code and docs in both Smith and the sibling
   Temper checkout.
